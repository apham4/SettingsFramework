UE5 plugin Settings Framework for game settings management with skeletal settings screen widgets using CommonUI and EnhancedInput.

Widgets support KBM and Gamepad.

Engine Version: Unreal Engine 5.7

Example host project: https://github.com/apham4/SFHost

Documentation: https://apham4.github.io/SettingsFramework/ 

Setup Guide: https://apham4.github.io/SettingsFramework/installation/

C++ API Reference: https://apham4.github.io/SettingsFramework/api/
